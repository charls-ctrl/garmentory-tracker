Connect Google Drive

This links your tracker to a file in your Google Drive. Your progress auto-saves every time you check a step. Sunil and Sam get a read-only link that always shows your current state.

You need a free Google Cloud Client ID. The setup guide below walks through every step — takes about 15 minutes once.

1
Go to console.cloud.google.com Sign in with your Google account. Click "Select a project" at the top → "New Project" → name it Garmentory Tracker → Create.
2
Enable the Google Drive API In the left sidebar: APIs & Services → Library → search "Google Drive API" → click it → Enable.
3
Configure the OAuth consent screen APIs & Services → OAuth consent screen → External → Create. Fill in App name: Garmentory Tracker. User support email: your email. Developer contact: your email. Save and continue through all screens. No scopes needed — just save and continue.
4
Create OAuth credentials APIs & Services → Credentials → + Create Credentials → OAuth client ID → Application type: Web application → Name: Garmentory Tracker Web.

Under "Authorised JavaScript origins" click + Add URI and enter exactly:
http://localhost:8000

Leave "Authorised redirect URIs" empty. Click Create. Copy your Client ID from the popup.
5
